window对象

BOM 的核心是 window 对象, 表示浏览器的实例, window 对象在浏览器中有两重身份, 一个是ECMAScript 中的 Global 对象, 另一个就是浏览器窗口的 JavaScript 接口

Global 作用域

window 对象被复用为 ECMAScript 的 Global 对象, 所以通过var声明的全局变量和全局方法都会变成window的属性和方法
