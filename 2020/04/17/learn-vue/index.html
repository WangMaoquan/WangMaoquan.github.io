我们需要关注的是一下两个文件shims-tsx.d.ts shims-vue.d.ts

  • shims-tsx.d.ts，允许你以.tsx结尾的文件，在Vue项目中编写jsx代码
  • shims-vue.d.ts 主要用于 TypeScript 识别.vue 文件，Ts默认并不支持导入 vue 文件，这个文件告诉ts 导入.vue 文件都按VueConstructor处理。
